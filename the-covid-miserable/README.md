# The Covid: Miserable

Date: 15-11-2021

A glimpse to Coronavirus 2019 in Malaysia and Global. Data span across 2020 to 2021 encompassing attributes Tests, Cases, Deceased, Vaccination and Recovered were primarily used from numerous open source appropriate channel.

SEIR Model was implemented to simulate the outbreak. Logistic functions to project the inflection point and growth rate of the pandemic. Further Exploratory Data Analysis was done to understand the data. Abundance machine learning model were integrated for forward forecast. Hyper parameters tune was done and custom model was created.

The main objective to forecast how likely the pandemic is going to end. 

The project aim to Estimate the Picture of Covid-19.
